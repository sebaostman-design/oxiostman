# oxiostman.com

Статический сайт клинического психолога Оксаны Остман с админ-панелью Decap CMS.

## Структура

```
.
├── index.html              # Главная страница
├── admin/
│   ├── index.html          # Админ-панель (/admin/)
│   └── config.yml          # Конфигурация Decap CMS
├── content/
│   ├── settings.json       # Общие настройки (фото, цены, ссылки)
│   ├── ru.json             # Русский текст
│   └── en.json             # Английский текст
├── images/                 # Загрузки из админки
├── netlify.toml            # Конфигурация Netlify
└── README.md
```

## Как клиент редактирует контент

1. Зайти на `https://<домен>/admin/`
2. Войти по email + паролю (Netlify Identity)
3. Редактировать поля, нажать **Publish** → сайт пересоберётся автоматически через 1–2 минуты

## Локальная разработка

Нужен любой HTTP-сервер (fetch не работает на `file://`):

```bash
python3 -m http.server 8080
# или
npx serve .
```

Открыть `http://localhost:8080/`.

## Деплой

Push в `main` → Netlify автоматически деплоит.

## Форматирование текста

В полях заголовков:
- `*слово*` — курсивом розового цвета
- перенос строки — новая строка

Пример: `Профессиональный\n*путь*` → «Профессиональный» обычным, «путь» курсивом ниже.
